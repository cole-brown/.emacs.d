** spotify:

I chose 'git subtree' for this because:
  https://codewinsarguments.co/2016/05/01/git-submodules-vs-git-subtrees/


Command:

git subtree pull --prefix lib/spotify git@github.com:danielfm/spotify.el.git master --squash
